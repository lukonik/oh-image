## What is Oh Image?

<span className="project-title pr-1">Oh Image</span> is a modern image optimization library for Vite and React applications that automatically handles responsive images, placeholder generation, and performance optimization. It eliminates the manual work of creating multiple image sizes and formats, while providing a seamless developer experience.

Think of it as the Next.js Image component, but built specifically for the Vite ecosystem.

## Key Features

- **Automatic Image Optimization** - Converts and optimizes images using Sharp during build time
- **Responsive Images** - Generates multiple breakpoints and srcsets for different screen sizes
- **Blur Placeholders** - Creates low-quality image placeholders for smooth loading experiences
- **Format Conversion** - Automatically converts images to modern formats like WebP
- **React Component** - Drop-in `<Image>` component with priority loading and fill mode support

## Components

<span className="project-title pr-1">Oh Image</span> consists of two main parts:
* **React Image Component** - Optimized image component with built-in responsive and placeholder features
* **Vite Plugin** - Build-time image optimizer that processes images automatically
